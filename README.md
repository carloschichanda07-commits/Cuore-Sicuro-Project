# Cuore Sicuro

**Cuore Sicuro** è una piattaforma web dedicata alla prevenzione del cyberbullismo e alla gestione sicura delle segnalazioni. 

Questo progetto nasce dall'esigenza di creare un ambiente digitale protetto e sicuro, con una struttura scalabile pensata per istituzioni scolastiche.

## 🚀 Status del Progetto
Il sistema è stato interamente ideato, sviluppato e venduto all'**I.I.S. Gastaldi-Abba** (Genova) nel 2026, dove è stato implementato con successo per supportare le attività di prevenzione contro il cyberbullismo.

*Nota: Questa repository contiene una versione "showcase" del codice, ottimizzata per finalità di portfolio e revisione tecnica.*

## 🛠 Tecnologie Utilizzate
Il progetto è stato sviluppato adottando uno stack solido e professionale:

- **Backend:** PHP (logica di business e gestione dati)
- **Database:** SQL (archiviazione sicura delle segnalazioni)
- **Frontend:** HTML5, CSS3, JavaScript (interfaccia utente responsive e intuitiva)
